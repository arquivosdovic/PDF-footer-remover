# PDF-footer-remover
Utilitário Web para limpeza cirúrgica de PDFs. Remove rodapés intrusivos, numeração de páginas e marcas d'água de exportadores de IA ajustando as coordenadas de MediaBox e CropBox via Python (Pikepdf). Diferente de máscaras visuais, esta solução elimina a interação com os dados da área afetada, garantindo um copiar e colar limpo.
